# Investment Analyst Agent

An AI-powered Investment Analyst built for the IBM SkillsBuild Hackathon using IBM Orchestrate and IBM Granite Models.

## Overview

The Investment Analyst Agent automates the investment analysis process by gathering financial data, performing sequential analysis, evaluating market sentiment, assessing investment risks, and generating AI-powered Buy, Hold, or Sell recommendations.

## Features

- Financial Data Analysis
- KPI Extraction
- Financial Ratio Calculation
- Competitor Comparison
- Market Sentiment Analysis
- Risk Assessment
- AI Investment Recommendation
- Interactive Dashboard
- Responsive Premium UI

## Technologies

- HTML5
- CSS3
- JavaScript
- Chart.js
- IBM Orchestrate
- IBM Granite Models
